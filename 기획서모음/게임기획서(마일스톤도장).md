# UI 그래픽 — 마일스톤 도장 자산 (게임 빅 모먼트용)

이미 만든 승소/패소 도장 스타일을 기반으로 게임 내 큰 마일스톤(승진·결혼·박사·엔딩 등)에 발동되는 도장 자산. 동일한 `showStampEffect()` 시스템으로 적용.

---

## 0. 공통 디자인 컨셉

승소/패소 도장과 동일한 형식:
- **사각 도장** (살짝 비뚤어진 모서리)
- **한글 텍스트** 가운데 큰 글씨
- **약간 회전** (-5도 ~ +5도)
- **그런지 텍스처** (낡은 도장 효과, 잉크 갈라짐)
- **투명 배경** (PNG with alpha)
- **사이즈**: 1024×1024 (PIL trim·512 리사이즈는 나중에 자동)

색만 마일스톤별로 차별화.

### 공통 키워드 (모든 도장 프롬프트에 자동 적용)
```
square ink stamp, korean modern office stamp aesthetic,
crisp ink texture with slight imperfection on edges,
slightly rotated, transparent PNG background,
no white background, no black background,
1024x1024 pixel
```

### 공통 Negative Prompt
```
no realistic photo, no 3d render, no calligraphy brush style,
no joseon traditional pattern, no scroll, no white fill,
no black fill, blurry, low quality
```

## 만들 때 권장사항

### 1. 승소/패소 도장 스타일 일관 유지
- 같은 사각 형태
- 비슷한 그런지 텍스처 강도
- 비슷한 회전 각도 (-5° ~ +5°)
- 한글 텍스트 폰트 톤 통일

### 2. 투명 배경 필수
이미 흰/검은 배경 도장은 PIL로 후처리 가능하지만 **투명 배경으로 만들면 처리 단계 생략**.

### 3. AI 도구 활용
- **Midjourney**: `--ar 1:1 --style raw` 추가, V6 이상
- **DALL-E 3**: "transparent background" 키워드 강조
- **Stable Diffusion**: 위 negative prompt 적극 활용

### 4. 후처리 (필요 시)
PIL 자동 처리 코드:
```python
from PIL import Image
img = Image.open('승진.png').convert('RGBA')
data = list(img.getdata())
new_data = []
for r, g, b, a in data:
    # 흰 배경 투명화
    if r > 220 and g > 220 and b > 220:
        new_data.append((r, g, b, 0))
    else:
        new_data.append((r, g, b, a))
img.putdata(new_data)
img.save('승진.png')
```

### 5. 톤 일관성 체크
4~7장 만든 후 한 폴더에서 비교:
- 색상 균형 (모두 같은 채도/명도?)
- 텍스처 강도 (그런지 양 비슷?)
- 회전 각도 (다양하지만 ±5° 안?)
- 가장자리 형태 (모두 사각? 라인 두께 비슷?)

이질적이면 같은 시드로 재생성.

## 적용 코드 (이미 게임에 구현됨)

```js
// game.html에 이미 정의됨 (line ~5160)
function showStampEffect(type) {
  const stampMap = {
    win: 'UI/이펙트/승소.png',
    lose: 'UI/이펙트/패소.png',
    // 새 자산 추가 시 여기 매핑 추가
    promotion: 'UI/이펙트/승진.png',
    phd: 'UI/이펙트/박사.png',
    ending: 'UI/이펙트/엔딩.png',
    marriage: 'UI/이펙트/결혼.png',
    elected: 'UI/이펙트/당선.png',
    birth: 'UI/이펙트/출산.png',
    mainCleared: 'UI/이펙트/사건종결.png',
  };
  // ... (이펙트 발동)
}
```

자산 PNG만 폴더에 추가하면 코드 수정 없이 발동 가능 (또는 매핑 한 줄 추가).
